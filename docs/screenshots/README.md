## How to capture terminal screenshots on CAINE OS / Ubuntu

```bash
# Install scrot (lightweight screenshot tool)
sudo apt install scrot

# Capture full screen
scrot docs/screenshots/01_banner.png

# Or use the GNOME screenshot tool
gnome-screenshot -f docs/screenshots/01_banner.png
```
